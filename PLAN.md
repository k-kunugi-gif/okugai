## 1. 事業概要

AIキャラクターが運営する「検証・教育型」FXオンラインサロンを立ち上げ、
**①有料サロン会員（Discord+Stripe自前決済）②インジケータ/ツール販売 ③投資スクール（バックエンド）** の3本柱で収益化する。
FX口座開設アフィリエイトは姉妹プロジェクト **~/fx-media（海外FX比較メディア）** が担い、相互送客する。

- **運営形態**: キャラクター運営（顔出しなし・AIキャラ）。VTuber事業（~/vtuber-business）のノウハウを流用
- **差別化軸**: 「AI×検証」。感覚や聖杯探しではなく、バックテスト・データ検証で手法のエッジを確かめる文化のサロン。自動売買開発（~/auto-invest）の実装力をコンテンツ化
- **ポジショニング**: シグナル配信サロンではなく **学習コミュニティ + 検証ツール + 教育**（→ 法規制の節を参照）

## 3. 法規制ガードレール（最重要・全コンテンツ共通ルール）

### 3.1 金融商品取引法（投資助言・代理業）

無登録での投資助言は刑事罰対象。以下を厳守する。

**やらないこと（レッドライン）**
- 「今ドル円ロング」等の個別・具体的な売買推奨、エントリー/イグジットのシグナル配信（有料・無料問わずサロン内では行わない）
- 会員個別の相談に対する具体的な売買指示
- 「このツールに従えば勝てる」型の断定的販売トーク
- インジケータ+個別サポートで実質シグナル配信になる商品設計

**商品ラインナップの法的整理（2026-07-11確定・同日改訂）**
- ⭕ サロン内**リアルタイムトレード公開(実況席)**: ユーザー決定により実施。ただし**観戦型の運用ルールを厳守**——①「入ります」は言っても「入ってください」等の指示・推奨表現は使わない ②毎配信に免責(売買判断は自己責任)を表示 ③必ず事後検証とセットにする ④個別の売買相談には応じない。※有償の先出し配信が無登録助言と認定された摘発例があるリスクは説明済み。運用ルールからの逸脱は厳禁
- ⭕ サロン会員は**ツールの使用権**を持つ(検証ツール・インジ)。EA・上位ツールは工房で**単品販売**(住民割引)。運用代行はしない
- ⭕ 億荘=**指導(育成)の場**: 勉強したい/稼ぎたい/極めたい人向けの集中指導。技術指導に限定し、個別の売買指示はしない
- ❌ MAM/PAMM: 投資運用業(一任)の登録が必要なため**サイト非搭載・保留**。やる場合は登録業者との提携が前提(将来フェーズ)
- ❌ シグナル配信(売買指示という形のもの)

**やること（セーフゾーン）**
- 手法の考え方・資金管理・検証方法の教育（一般的・抽象的な情報提供）
- 過去チャートを使った事後解説・検証結果の共有
- インジケータ/検証ツールの販売（判断は利用者自身が行う設計・表記）
- 会員同士が相場観を語り合う場の提供（運営者は個別助言しない）

### 3.2 景品表示法・広告表現
- 「月利◯%」「勝率◯%保証」「必ず勝てる」等の断定的・誇大表現は使わない
- 実績を出す場合は検証条件・期間・リスクを併記
- LP・すべての販売ページに投資リスク警告と「投資助言ではない」旨の免責を明記

### 3.3 特定商取引法
- Stripe自前決済 = 自分が販売事業者。**特商法表記には本名（または法人名）・住所・電話番号が必須**
- ⚠️ キャラクター運営でも特商法ページは実名/法人名になる点に注意。匿名性を保つなら**法人（合同会社等）名義+バーチャルオフィス**の検討を推奨
- 月額課金の解約方法・解約期限を明確に表示

### 3.4 ステマ規制（景表法・2023年10月〜）
- アフィリエイトリンクを含む投稿・記事には「PR」「広告」表記

```

   ↓
[リスト] LINE公式 or メール — フック: 無料インジケータ or 検証レポート
   ↓

   ↓

   ├─ ライト  980円/月 … 検証レポート閲覧・雑談ch
   └─ スタンダード 4,980円/月 … 全ch・月例検証会・ツール割引
   ↓
[バック] 億荘(オクソウ) — 選抜入居制の集中育成(家賃 月29,800円・全6部屋 ※仮)
   └─ 検証ツールを自作できるようになる実装指導(Python/MT4/MT5)+資金管理の型
       各入居者にDiscord個人部屋(スレッド)。卒業要件を満たすと「億荘◯号室 卒業生」
       ※サロンで信頼構築後、6ヶ月目以降に入居募集開始。詳細はREQUIREMENTS.md §2.5
```

- アフィ収益は fx-media 側で計上（口座開設1件2〜4万円）。サロン内では「PR」表記の上で口座紹介

## 5. 技術構成（Discord + Stripe 自前）

```
LP (静的HTML → 後にホスティング)
 └─ Stripe Checkout (subscription)
     └─ webhook → 会員管理bot (Node/Cloudflare Workers等)
         ├─ 決済成功 → Discord招待リンク発行 + 会員ロール付与
         ├─ 支払い失敗/解約 → ロール剥奪
         └─ 顧客ポータル(Stripe Billing Portal)で解約自己完結
Discord サーバー構成:
 #announcements / #検証レポート / #ツールサポート / #雑談 / #質問(教育のみ)
```

- 決済手数料はStripe 3.6%のみ。DMM/CAMPFIRE(10〜20%)比で圧倒的に有利
- インジケータ販売は Stripe Payment Links（単発決済）+ 購入者限定chでサポート
