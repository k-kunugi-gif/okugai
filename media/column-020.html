COLUMN VOL.020

相場には時間割がある。
——東京・ロンドン・NYの歩き方

基礎・入門読了目安 9分2026.07 更新
ブリッツ SCALP / 短期・執行担当

秒と分を刻む、億街のスプリンター。本コラムは億街の住人が執筆し、運営が内容を検証・監修しています。

KEY POINTS — この記事でわかること
  • 東京・ロンドン・NYで相場の性格がどう変わるか
  • 兼業の主戦場=22〜25時のゴールデンタイム
  • 触ってはいけない時間帯(早朝)とその理由

ブリッツだ。今日は「いつ戦うか」の話。

同じドル円でも、朝9時と夜23時では別の生き物だ。値動きの速さも、素直さも、スプレッドもまるで違う。手法より先に時間帯を決めろ——これが俺の持論だ。生活リズムに合わない時間帯の手法は、どんなに優秀でも続かない。

3大市場の時間割——世界は8時間ずつ交代で動く

為替市場は24時間動くが、主役は交代制だ。東京時間(9〜15時)は仲値(9:55)前後に動きが出るものの、全体としては静かなレンジになりやすい。ロンドン時間(17時〜)で欧州勢が参戦すると値動きの質が変わり、東京のレンジを壊すトレンドが出やすくなる。そしてNY時間(22時〜)——米指標の発表が重なり、1日で最も太い値動きが生まれる。

兼業の主戦場——22〜25時のゴールデンタイム

ロンドンとNYが重なる22〜25時(冬時間)は、参加者が最も厚く、トレンドが素直に伸びやすい3時間だ。そして都合のいいことに、日本の兼業トレーダーが帰宅してチャートの前に座れる時間と完全に重なる。

「仕事があるから不利」と思っている人は逆だ。専業が1日中画面を見て疲弊する一方、兼業は世界で一番おいしい3時間だけを、毎日同じルーティンで戦える。記録も同じ時間帯で揃うから、検証の質も上がる。

触ってはいけない時間帯

  • 早朝5〜8時(オセアニア〜東京前)——参加者が薄く、スプレッドが数倍〜数十倍に開く(スプレッドの回)。値が飛んでも追随する参加者がいない
  • 金曜深夜・週末クローズ間際——週をまたぐポジション調整で不規則に動く。月曜朝の窓開けリスクも」
  • 大物指標の前後30分——時間帯というより「時刻」の地雷。指標の回の通り、カレンダーで避ける
POINT — 時間帯を固定する3つの利点

①相場の「性格」が揃うので検証データが濁らない ②生活リズムに組み込めるので続く ③同じ時間の値動きを見続けることで、異変(いつもと違う動き)に気づける——時間を固定することは、変数を1つ消すことだ。

時間帯の名物イベント——仲値・フィキシング・オプションカット

時刻(日本時間)イベント何が起きやすいか
9:55東京仲値実需のドル買いで、仲値に向けてドル円が堅くなる日が多い(五十日は特に)
24:00(冬)ロンドンフィキシング欧州の値決め。月末は特殊なフローで荒れやすい
24:00(冬)NYオプションカット大きなオプション設定水準に価格が吸い寄せられる磁石効果が語られる

これらは「必ずこう動く」ではなく、「参加者が意識する時刻」だ。知っていれば、理由の分からない動きに「ああ、フィキシングか」と説明が付く——説明が付くだけで、狼狽エントリーは大きく減る。

夏時間・冬時間の対応表——年2回の切替を忘れない

米国の夏時間(3月第2日曜〜11月第1日曜)には、すべてが1時間前倒しになる。ゴールデンタイムは21〜24時、NY指標は21:30発表。切替直後の1〜2週間は「いつもの時間に指標が来ない」混乱が起きがちなので、3月と11月はカレンダーの時刻表示を必ず確認しよう。ちなみに欧州の夏時間切替は米国と数週ズレるため、3月後半と10月末には英欧米の時差が変則になる「谷間の週」が存在する——豆知識だが、指標時刻のミスはこの週に集中する。

CHECKLIST — 時間帯を決める3項目

自分が毎日座れる時間帯を選んだ。その時間帯の名物イベントを知っている。夏冬時間の切替月(3月・11月)を把握した。全部YESになるまで、次へ進まないこと。

曜日のリズム——週の中にも時間割がある

1日の時間割の外側に、もう一つのリズムがある——曜日だ。

曜日性格注意点
月曜窓開けの可能性。午前は方向感が出にくい週末ニュースの消化待ち。朝イチの飛び乗りは慎重に
火〜木曜フローが安定し、テクニカルが素直に効きやすい週の主戦場。指標の配置だけ確認
金曜週末前の手仕舞いで、夕方以降は逆方向の調整も雇用統計の週は別物。持ち越し判断を22時までに

「なぜか月曜は成績が悪い」——記録を付けると多くの人がこれに気づく。原因は腕ではなく、方向感のない時間に普段と同じ回数を張っていることだったりする。曜日別の集計は、記録の集計に1列足すだけでできる。

時間帯×手法の相性マトリクス

手法タイプ東京ロンドンゴールデンタイム
レンジ逆張り◎ 主戦場△ ブレイクに轢かれる× トレンドに逆らう形
トレンドフォロー△ 続かない○ 初動を掴む◎ 最も伸びる
ブレイクアウト× ダマシ多発◎ 東京レンジの破壊を狙う○ 指標後の二段目

手法の成績が振るわないとき、疑うべきは手法そのものより先に「その手法を、性格の合わない時間帯で使っていないか」だ。レンジ逆張りをゴールデンタイムで回して「勝てない」と嘆くのは、雨の日にバーベキューをして天気に文句を言うのに似ている。

ゴトー日(五十日)と仲値——東京の実需フロー

東京時間の名物・仲値(9:55)をもう一歩深掘りする。5日・10日・15日…と5と10のつく日は「ゴトー日」と呼ばれ、企業の決済が集中する。輸入企業のドル買い需要が仲値に向けて膨らむため、「ゴトー日の仲値前はドル円が堅い」という経験則が昔から語られている。

ただし、これも「必ず」ではなく傾向だ。億街流の扱い方は2つ——①この時間帯の不自然な上昇を「実需フローかもしれない」と説明できるようになる(狼狽しない) ②手法にするなら、30回の検証にかけてから。有名なアノマリーほど、知っているだけの人と検証した人の差が出る。

自分の時間帯を見つける——1週間の観察ドリル

  1. 月〜金の5日間、自分が座れる時間帯を固定して観察する(取引しなくていい。チャートを開くだけ)
  2. 毎日、その時間帯の値動きを3行でメモ——「レンジ幅◯pips」「◯時に動意」「スプレッドの様子」
  3. 週末に5日分を読み返し、「この時間帯はどんな手法が合いそうか」を1行で結論する
  4. 翌週から、その仮説を1,000通貨で試す

この1週間の観察は、どんな教材より「あなたの時間帯」の解像度を上げる。相場は毎日開いている——急いで全部を知ろうとせず、自分の3時間を深く知る方が、結局早い。

専業と兼業——時間戦略はこう違う

同じ時間割でも、立場によって戦略は変わる。兼業の武器は「時間の制約」そのものだ——毎日同じ2時間しか見られないことが、検証データの均質性と生活リズムを守ってくれる。逆に専業の敵は「全部の時間帯が見えてしまう」こと。東京で少し取り、ロンドンで少し返し、NYで疲れ切って判断を誤る——時間の自由が規律を溶かす。

だから専業にも「コアタイム宣言」を勧める。主戦場を2〜3時間に区切り、それ以外は検証・記録・休息に充てる。市場に長く座ることと、市場から多く取ることは、まったく別の話だ——これは専業になった住人が最初に学ぶ教訓でもある。

早朝スキャルの現実——「おいしい」と「難しい」は別

早朝(5〜8時)の値飛びを狙うスキャル手法が時々話題になる。流動性が薄い時間の歪みを取る発想自体には合理性があるが、現実の壁を並べておく——①スプレッドが平常の数倍〜数十倍でコスト計算が崩れやすい ②約定拒否・滑りが頻発する ③業者側も対策済みで、歪み自体が年々小さい。

つまり「歪みがある」ことと「個人がコストを払って取れる」ことの間には深い川がある。初心者がこの時間帯に手を出す合理性は、正直ほぼない。おいしい話ほど、コスト構造から検算する——時間帯選びにも、この街の基本動作は変わらない。

よくある質問

初心者はどの時間帯から始めるべきですか？

ロンドン初動(17〜19時)かゴールデンタイム(22〜25時)を推奨します。参加者が厚く値動きが素直で、スプレッドも安定しています。まず1つの時間帯に固定して30トレードの記録を作ってください。

東京時間は稼げないのですか？

稼げないのではなく性格が違います。レンジになりやすいため、レンジ逆張りや仲値(9:55)前後のフローを狙う手法が向きます。トレンドフォロー手法をそのまま持ち込むと機能しにくい、という相性の問題です。

夏時間と冬時間で何が変わりますか？

欧米の市場オープンが1時間早まります(ロンドン16時〜、NY21時〜、ゴールデンタイム21〜24時)。3月と11月の切替時期は、いつもの時間感覚とズレるので指標時刻も含めて要確認です。

24時間ずっとチャンスを探すべきですか？

逆効果です。時間帯によって相場の性格が違うため、全部を追うと検証データが濁り、生活も壊れます。「自分の時間帯」を決めて、そこだけで記録を積む方が上達は速くなります。
