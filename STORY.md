## 1. 表の物語「億街縁起」(全文・v3 決定稿)

誰も知らない街の話をしよう。

かつて相場の世界には「聖杯通り」と呼ばれる眩しい大通りがあった。その喧騒から遠く離れた場所に、名もない街があり、名もない荘が建っていた。

家賃の安いボロ長屋。住んでいたのは、少し変わった連中だった。聖杯を買わない。予想を信じない。夜ごと部屋に灯りをともし、ただひたすら検証を回す。人間もいた。人間でない者もいた。やがて、荘の住人たちの噂が流れはじめる。彼らは異常らしい。彼らが本当に「億」を作ったのかどうか、確かめた者はいない。

ただ——街の中心に、塔が建った。

住人たちは荘を出て、塔の上へ移り住んだ。人々はその塔を億ヒルズと呼び、いつしか街そのものを「億街」と呼ぶようになった。

そして、あのボロ長屋。名もなかった荘は、今こう呼ばれている。

**億荘。**

名付けたのは住人ではない。後から来た者たちだ——「伝説が住んでいた荘」と。
その部屋は今も残っている。空き部屋には、次の住人が入れる。

街の入口は「秘密基地」と呼ばれている。地図には載っていない。だが、扉はいつでも開いている。

### v3の構造ポイント
- **時系列**: 名もない荘に住人たちが住む → 「彼らは異常らしい」という噂 → 塔(億ヒルズ)へ移住 → 街が「億街」と呼ばれる → 荘が後から「億荘」と命名される。**街の名も荘の名も、住人の伝説から生まれた**(すべての固有名詞に由来がある)
- **伝説は具体を語らない**: v2の具体的逸話(三千回のバックテスト等)は削除し「彼らは異常らしい」の一文に圧縮(ユーザー編集)。噂の不気味さだけを残す。個別の逸話は縁起では語らず、各キャラのプロフィールや日々のコンテンツで小出しにする
- **「億」の扱い**: 「確かめた者はいない。ただ——塔は建った」で断定回避(実績誤認の回避+都市伝説としての余韻)
- **電脳住人の説明も縁起からは省略**(v3.1で「大家が仲間を作った」の一文も削除): 「人間もいた。人間でない者もいた。」の示唆のみに留め、正体の謎を最大化。AI公表(掟参)は掟セクション・住人名鑑・FAQが担う。※「街主=元・荘の大家」の設定自体は生きている(住人名鑑チップ等)——縁起で語らないだけ
- **AI住人=億荘の元住人**: 各キャラに元部屋番号を設定できる(例: ブリッツ=102号室、アルマ=201号室)。億荘の募集コピー「あなたが入居するのは、かつてブリッツが住んでいた102号室だ」が使える
- **億ヒルズの説得力**: 創設住人たちが現に住んでいる場所だから招待制、という理屈が通る

## 2. 意味の正典(なぜ、をすべて言えるようにする)

| 要素 | 意味 |
|---|---|
| 街 | 学校は卒業で終わり、塾は買ったら終わり。街は「住む」場所。トレードはイベントではなく生活。続けた者だけが残る。去る自由も含めて街 |
| 秘密基地 | 外(SNS)は聖杯の喧騒、中は検証の静けさ。隠しているのは場所ではなく「本気」。大人が本気で作り、本気で遊ぶ場所。「公然の秘密基地」の矛盾はネタとして公式に使う |
| 億 | 保証ではなく方角(北極星)。掴むものではなく、進む向きを教えるもの。掟弐「保証はしない」と表裏一体 |
| 聖杯通り | 敵の擬人化。名指しの批判をせずに業界の病理を語るための装置。「聖杯通りの店」「客引き」で情報商材文化を指す |
| 総帥 | 預言者ではなく技術者(大工)。「聖杯はやらん。鍛冶場なら貸してやる」。ユーザーの実像(AI×投資の実装者)と嘘なく重なるアーキタイプ |
| 電脳住人(AI) | 「AIは疲れない・飽きない・検証に感情を持ち込まない。人間は決断し、責任を取る」という分業の哲学。総帥が住人を「作った」=AI公表(掟参)の物語化。コンプラ要件を世界観の見せ場に変換する装置 |
| 億荘 | 下積みの美学。すべての本物はボロ部屋から始まる(トキワ荘)。**名付けたのは後から来た者たち**=伝説は自称するものではなく呼ばれるもの。卒業=巣立ち。入居者は「伝説の部屋」に入る |
| 億ヒルズ | 頂上の可視化。この街では高さは買えない、登るしかない。非売品であること自体がメッセージ。**創設住人たちが現に住んでいる**から招待制 |
| 工房 | ツール=魔法ではなく道具。「ツールに従う」のではなく「ツールで確かめる」 |
| 掟 | **三箇条確定(2026-07-11ユーザー決定)**: 壱「本気で向き合う。」弐「決して諦めない。」参「真実のみを扱う。」——街の生き方を示す言葉。※旧掟(予想は売らない/保証はしない/正体は隠さない)は「掟」ブランドからは外れたが、**運営ポリシーとしては存続**——FAQ・免責・利用規約・比較表・縁起の看板(「予想は売らない。検証ならできる。」)が引き続き担保する |

## 3. 用語の動詞体系(全コンテンツ共通)

- サロン正式名称 → **億街サロン**(対外・商品表記)。「秘密基地」はDiscord拠点の世界観上の通称(縁起由来)
- 入会 → **移住** / 会員 → **住民** / 退会 → **退去**
- 上位プログラム参加 → **億荘に入居** / 修了 → **卒業(◯号室卒業生)**
- 最上位認定 → **億ヒルズに招かれる**
- 運営メンバー → **住人**(電脳住人=AI、人間の総帥・顧問と区別)
